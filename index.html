O que são Clients?

No linguajar WEB, Client é o dispositivo que está consumindo a informação pela internet

Conceitos importantes
    Cache: Arquivos que foram baixados anteriormente ainda estão no computador, então o cache existe para que o computador não precise baixar todos os arquivos novamente
    Navegadores: São programas criados por empresas para abrir/executar arquivos, eles tamém são chamados de browsers, eles são gratuitos
    Aplicações WEB: São soluções que possuem a internet como meio de comunicação entre Client e Server não sendo necessário sua instalação (Aplicação WEB-> Acesso através do navegador / Aplicativo -> Instalado )
    Dispositivos Móveis: São os Notebooks, os Smartphones, os Tablets

O que são Servers?

O Server como o próprio nome induz tem a função de servir a informação para o client

Conceitos importantes
    Tipos de servidores:
      Arquivo
      Segurança
      Streaming
      E-mail
      WEB
    Hardware--Software

      Hardware é a parte física (Armazenamento; Memóriaa; Processador)

      Software é a parte não palpável do computador (Sistemas Operacionais; Monitoramento; Servidores WEB)

    Exemplos de Servidores WEB: Apache

    Exemplos de Sistemas Operacionais: Ubuntu; Oracle; Linux; Fedora

    Tipos de Servidores: Proxy; Firewall; WEB Server

    Hospedagem de Sites: Para que as pessoas possam acessar o seu site, é necessário que ele esteja hospedado em algum lugar. Diversas empresas disponibilizam planos de hospedagem (ex: Google Cloud; HostGator; Umbler)


Sobre Linguagens de Programação Web

Linguagens de programção
O que é?

É uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas.

Linguagens executadas do lado do Servidor: PHP; Ruby; Java; C#

Linguagens executadas do lado do Cliente: JavaScript

HTML não é Linguagem de Programação, HTML é um arquivo de marcação de textos delimitados por nomes que o navegador consegue interpretar. Cada nom tem uma função e comportamento específico

Falando sobre tags

As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. MAS nem todas precisam de fechamento, como é o caso da tag de imagem, que é "< img src="" alt="">" e a tag de input "< input type="text">"
